# Метод моментов — основная задача электростатики (М1)

## Описание задачи

Два металлических электрода поддерживаются при постоянной разности потенциалов $\Delta\varphi = V$.
Требуется найти распределение электрического поля между электродами, построить картину силовых
линий и рассчитать ёмкость системы.

Метод решения — **метод моментов (МоМ)**: поверхность каждого проводника разбивается на $N$
малых плоских панелей с неизвестными зарядами $q_i$. Потенциал центра панели $i$ от всех остальных
панелей (и от себя) равен заданному потенциалу проводника:

$$\sum_{j} P_{ij} \, q_j = \varphi_i^{\text{цель}}, \qquad
P_{ij} = \begin{cases}
\dfrac{1}{4\pi\varepsilon_0 |\mathbf{r}_i - \mathbf{r}_j|}, & i \ne j, \\[6pt]
\dfrac{2}{4\pi\varepsilon_0}\sqrt{\dfrac{\pi}{A_i}}, & i = j
\end{cases}$$

Самопотенциал ($i=j$) вычислен для кругового диска площадью $A_i$, равноценного по площади
квадратной или угловой панели. СЛАУ решается через `scipy.linalg.solve`.

## Входные параметры и их ограничения

### Параллельные пластины (`plates`)

| Параметр | Описание | Допустимые значения |
|----------|----------|---------------------|
| `--L` | Сторона квадратной пластины, м | $> 0$, разумно $0.1$–$10$ м |
| `--d` | Расстояние между пластинами, м | $0 < d < L$; при $d \to 0$ матрица плохо обусловлена |
| `--nx`, `--ny` | Число панелей по x и y | $\ge 2$; матрица $P$ имеет размер $2 n_x n_y \times 2 n_x n_y$; при $n > 30$ решение медленнее |
| `--V` | Разность потенциалов, В | любое ненулевое |

### Концентрические сферы (`spheres`)

| Параметр | Описание | Допустимые значения |
|----------|----------|---------------------|
| `--R_inner` | Радиус внутренней сферы, м | $> 0$ |
| `--R_outer` | Радиус внешней сферы, м | $> R_\text{inner}$ |
| `--n_theta` | Широтных полос | $\ge 4$ |
| `--n_phi` | Долготных полос | $\ge 8$, рекомендуется $\approx 2 n_\theta$ |
| `--V` | Разность потенциалов, В | любое ненулевое |

## Выходные параметры и визуализация

Программа выводит в консоль:

- Численную ёмкость системы $C_\text{МоМ}$
- Аналитическое значение (если доступно) и относительную погрешность
- Полные заряды на каждом электроде и их сумму (проверка нейтральности)

Сохраняется PNG-файл с графиком.

**Параллельные пластины** — три панели:
1. Карта потенциала в сечении $y=0$ (закрашенные контуры + изолинии)
2. Силовые линии электрического поля (streamplot, цвет = $\ln(1+|E|)$)
3. Поверхностная плотность заряда $\sigma(x)$ вдоль средней линии каждой пластины — хорошо виден краевой эффект

**Концентрические сферы** — два панели:
1. Карта потенциала в плоскости $y=0$ с маскировкой внутри сфер
2. Силовые линии поля

## Способ запуска

### Веб-интерфейс

```bash
chmod +x run.sh          # один раз (если не сделано)
./run.sh install         # установить зависимости
./run.sh server          # запустить сервер
```

Откроется по адресу **http://localhost:5001**

Если порт уже занят:

```bash
lsof -ti:5001 | xargs kill -9   # освободить порт
PORT=5002 ./run.sh server        # или запустить на другом
```

### Тесты

```bash
./run.sh tests -v
```
