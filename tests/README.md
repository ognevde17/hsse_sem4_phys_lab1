## Аналитические основы метода моментов

Поверхность каждого проводника разбивается на $N$ плоских панелей. Панель $j$ несёт заряд $q_j$, сосредоточенный в её центре (приближение точечного заряда для внедиагональных элементов). Потенциал в центре панели $i$:

$$\varphi_i = \sum_{j=1}^{N} P_{ij}\, q_j$$

Коэффициенты матрицы:

$$P_{ij} = \begin{cases}
\dfrac{1}{4\pi\varepsilon_0}\cdot\dfrac{1}{|\mathbf{r}_i - \mathbf{r}_j|}, & i \ne j,\\
\dfrac{1}{4\pi\varepsilon_0}\cdot 2\sqrt{\dfrac{\pi}{A_i}}, & i = j.
\end{cases}$$

Диагональный элемент $P_{ii}$ — самопотенциал равномерно заряженного кругового диска площадью $A_i$, вычисляемый аналитически как $\frac{1}{2\varepsilon_0}\sqrt{\frac{A_i}{\pi}} \cdot \frac{1}{A_i} = \frac{1}{4\pi\varepsilon_0}\cdot 2\sqrt{\pi/A_i}$.

Требуя $\varphi_i = \varphi^{\text{цель}}$ на каждой панели, получаем СЛАУ $P\,\mathbf{q} = \boldsymbol{\varphi}^{\text{цель}}$, решаемую через `scipy.linalg.solve`.

Для изолированного конденсатора (два проводника, нет внешних зарядов) дополнительно накладывается условие электронейтральности $\sum_i q_i = 0$. Оно вводится методом проекции: сначала решается система без ограничений $P\mathbf{q}_0 = \boldsymbol{\varphi}$, затем:

$$\mathbf{q} = \mathbf{q}_0 - \lambda\,\mathbf{p}, \quad \mathbf{p} = P^{-1}\mathbf{1}, \quad \lambda = \frac{\sum_i q_{0i}}{\sum_i p_i}.$$

## Структура тестов

### `TestIsolatedSphere` — одиночная сфера

**Аналитическое решение задачи Дирихле для шара.**

Сфера радиуса $R$ — идеальный проводник при потенциале $V$. По теореме Гаусса поле вне сферы совпадает с полем точечного заряда $Q$:

$$\mathbf{E}(r) = \frac{Q}{4\pi\varepsilon_0 r^2}\hat{\mathbf{r}}, \quad r > R.$$

Потенциал вне сферы:

$$\varphi(r) = \frac{Q}{4\pi\varepsilon_0 r}, \quad r \ge R.$$

На поверхности $r = R$:

$$V = \varphi(R) = \frac{Q}{4\pi\varepsilon_0 R} \implies C = \frac{Q}{V} = 4\pi\varepsilon_0 R.$$

Численные значения: $R = 0.1$ м $\Rightarrow C \approx 11.1$ пФ; $R = 0.3$ м $\Rightarrow C \approx 33.4$ пФ; $R = 0.5$ м $\Rightarrow C \approx 55.6$ пФ.

**Тесты:**
- `test_capacitance` — $C_\text{МоМ}$ совпадает с $4\pi\varepsilon_0 R$ с относительной погрешностью до 3% для трёх значений $R$.
- `test_potential_outside` — потенциал в трёх внешних точках ($r = 0.6,\,1.0,\,2.0$ м) совпадает с $Q/(4\pi\varepsilon_0 r)$ с точностью 2%.

### `TestNestedSpheres` — концентрические сферы

**Аналитическое решение методом разделения переменных / теоремой Гаусса.**

Пусть внутренняя сфера ($R_1$) при потенциале $+V/2$, внешняя ($R_2$) при $-V/2$.
По теореме Гаусса в зазоре поле сферически симметрично и радиально:

$$E(r) = \frac{Q}{4\pi\varepsilon_0 r^2}, \quad R_1 \le r \le R_2.$$

Интегрируя от $R_1$ до $R_2$:

$$V = \int_{R_1}^{R_2} E(r)\,dr = \frac{Q}{4\pi\varepsilon_0}\left(\frac{1}{R_1} - \frac{1}{R_2}\right) = \frac{Q(R_2 - R_1)}{4\pi\varepsilon_0 R_1 R_2}.$$

Ёмкость:

$$C = \frac{Q}{V} = \frac{4\pi\varepsilon_0 R_1 R_2}{R_2 - R_1}.$$

Потенциал в зазоре как функция $r$:

$$\varphi(r) = \frac{Q}{4\pi\varepsilon_0}\left(\frac{1}{r} - \frac{1}{R_2}\right) + \varphi(R_2), \quad R_1 \le r \le R_2.$$

По теореме об оболочке все панели на одной сфере несут одинаковую поверхностную плотность заряда $\sigma = Q / (4\pi R^2)$.

Вне обеих сфер при $Q_\text{total} = 0$ монопольный член исчезает. Ведущим является квадрупольный член $\sim Q R_2^2 / r^3$, который при $r = 20 R_2$ равен примерно $10^{-8}$ В — пренебрежимо мало.

Численные значения: $R_1 = 0.3$ м, $R_2 = 0.6$ м $\Rightarrow C \approx 66.8$ пФ; $R_1 = 0.2$ м, $R_2 = 0.8$ м $\Rightarrow C \approx 29.7$ пФ; $R_1 = 0.5$ м, $R_2 = 1.0$ м $\Rightarrow C \approx 111.3$ пФ.

**Тесты:**
- `test_capacitance` — относительная погрешность МоМ от аналитики не превышает 4% для трёх пар $(R_1, R_2)$.
- `test_charge_neutrality` — $\sum q_i < 10^{-10}$ Кл; нейтральность обеспечивается методом проекции.
- `test_equal_opposite_charges` — $|Q_1 + Q_2| / \max(|Q_1|, |Q_2|) < 5\%$; при проекции условие выполняется точно.
- `test_uniform_charge_distribution` — коэффициент вариации плотности $\sigma_i = q_i / A_i$ не превышает 10%. При равноплощадной параметризации (равномерно по $\cos\theta$) все панели имеют одинаковую площадь, однако центры панелей расположены неравномерно по углу, что вносит остаточную вариацию ~7–8% при 648 панелях.
- `test_far_field_potential_vanishes` — при $r = 20 R_2$ потенциал $< 5 \times 10^{-4}$ В, что соответствует подавлению монопольного члена при $Q_\text{total} = 0$.

### `TestParallelPlates` — параллельные пластины

**Аналитическое решение для бесконечных пластин и поправка Кирхгофа.**

Для бесконечных пластин площадью $S$ при расстоянии $d$ поле равномерное: $E = V/d$, ёмкость:

$$C_\infty = \frac{\varepsilon_0 S}{d} = \frac{\varepsilon_0 L^2}{d}.$$

Для конечных квадратных пластин $L \times L$ краевые эффекты увеличивают ёмкость. Поправка Кирхгофа первого порядка (из конформного отображения полуплоскости):

$$C \approx C_\infty \left[1 + \frac{d}{\pi L}\left(\ln\frac{\pi L}{d} + 1 + \frac{1}{2}\ln 2 + \ldots\right)\right].$$

При $L/d = 5$ поправка составляет около 15–25%; при $L/d = 20$ — около 5–10%. МоМ воспроизводит это превышение $C > C_\infty$.

Поверхностная плотность заряда на краях пластин выше, чем в центре — это физический краевой эффект, хорошо видимый на третьем графике визуализации.

Численные значения: $L = 1$ м, $d = 0.2$ м $\Rightarrow C_\infty \approx 44.3$ пФ, $C_\text{МоМ} \approx 67.6$ пФ ($C/C_\infty \approx 1.53$).

**Тесты:**
- `test_capacitance_large_aspect_ratio` — при $L/d = 20$ отношение $C_\text{МоМ}/C_\infty \in (1.0,\; 1.55)$.
- `test_charge_neutrality` — $\sum q_i = 0$ точно (метод проекции).
- `test_charge_sign` — суммарный заряд пластины при $+V/2$ положителен, при $-V/2$ — отрицателен.
- `test_capacitance_scales_with_area` — при увеличении $L$ в $\sqrt{2}$ раза (площадь в 2 раза) ёмкость растёт в $1.7$–$2.3$ раза, что соответствует закону $C \propto L^2/d$ с поправкой на краевые эффекты.

### `TestPotentialMatrix` — свойства матрицы $P$

**Математические свойства, следующие из физики.**

Матрица $P$ является матрицей Грама электростатического взаимодействия и обладает следующими свойствами:

**Симметрия** $P_{ij} = P_{ji}$: следует из принципа взаимности Грина — потенциал в точке $\mathbf{r}_i$ от единичного заряда в $\mathbf{r}_j$ равен потенциалу в $\mathbf{r}_j$ от единичного заряда в $\mathbf{r}_i$.

**Положительная определённость**: вытекает из положительности электростатической энергии $W = \frac{1}{2}\mathbf{q}^T P\,\mathbf{q} > 0$ для любого ненулевого $\mathbf{q}$. Гарантирует единственность решения СЛАУ.

**Диагональное доминирование по знаку**: самопотенциал $P_{ii} = \frac{1}{4\pi\varepsilon_0}\cdot 2\sqrt{\pi/A_i}$ растёт как $A_i^{-1/2}$ при уменьшении площади панели. При достаточно мелком разбиении $P_{ii}$ превышает сумму всех внедиагональных элементов в строке $i$, что соответствует физическому факту: собственная энергия локализованного заряда выше его взаимодействия с удалёнными зарядами.

**Тесты:**
- `test_symmetry` — $\|P - P^T\|_\infty < 10^{-12}$.
- `test_positive_definite` — минимальное собственное значение $\lambda_{\min} > 0$.
- `test_diagonal_dominance_sign` — для каждой строки $P_{ii} > \max_{j \ne i} P_{ij}$.
