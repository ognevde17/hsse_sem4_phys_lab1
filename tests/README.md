# Тесты

Запуск: `pytest tests/test_analytical.py -v` из корня проекта.

---

## Структура тестов

### `TestIsolatedSphere` — одиночная сфера

**Аналитика.**
Сфера радиуса $R$ с зарядом $Q$ создаёт снаружи потенциал точечного заряда:

$$\varphi(r) = \frac{Q}{4\pi\varepsilon_0 r}, \quad r \ge R$$

При $r = R$: $\varphi = \frac{Q}{4\pi\varepsilon_0 R}$, откуда ёмкость:

$$C = \frac{Q}{\varphi} = 4\pi\varepsilon_0 R$$

Для $R = 0.3$ м: $C = 4\pi \cdot 8.854 \cdot 10^{-12} \cdot 0.3 \approx 33.4$ пФ.

**Тесты:**
- `test_capacitance` — $C_\text{МоМ}$ совпадает с $4\pi\varepsilon_0 R$ с точностью 3% для трёх значений $R$
- `test_potential_outside` — потенциал во внешних точках соответствует формуле $K_E Q / r$ с точностью 2%

---

### `TestNestedSpheres` — концентрические сферы

**Аналитика.**
Из теоремы Гаусса поле между сферами радиальное:

$$E(r) = \frac{Q}{4\pi\varepsilon_0 r^2}, \quad R_1 \le r \le R_2$$

Разность потенциалов:

$$V = \int_{R_1}^{R_2} E \, dr = \frac{Q}{4\pi\varepsilon_0}\left(\frac{1}{R_1} - \frac{1}{R_2}\right)
    = \frac{Q(R_2 - R_1)}{4\pi\varepsilon_0 R_1 R_2}$$

Ёмкость:

$$C = \frac{Q}{V} = \frac{4\pi\varepsilon_0 R_1 R_2}{R_2 - R_1}$$

Для $R_1 = 0.3$ м, $R_2 = 0.6$ м: $C \approx 66.8$ пФ.

Потенциал в промежутке:

$$\varphi(r) = \frac{Q}{4\pi\varepsilon_0}\left(\frac{1}{r} - \frac{1}{R_2}\right) + \varphi(R_2)$$

**Тесты:**
- `test_capacitance` — точность 4% для трёх пар $(R_1, R_2)$
- `test_charge_neutrality` — сумма всех зарядов панелей равна нулю ($< 10^{-10}$ Кл)
    - `test_equal_opposite_charges` — относительный дисбаланс $|Q_1 + Q_2| / \max(|Q_1|, |Q_2|) < 5\%$; нейтральность обеспечивается методом проекции (два решения системы $P$, устойчивых к плохой обусловленности)
    - `test_uniform_charge_distribution` — коэффициент вариации плотности $\sigma_i = q_i / A_i$ не превышает 10%; при равноплощадной параметризации центры панелей остаются чуть неравномерными в угловом отношении, что вносит ~7–8% вариацию при 648 панелях
    - `test_far_field_potential_vanishes` — при $r = 20 R_2$ потенциал $< 5 \times 10^{-4}$ В; при $Q_\text{total} = 0$ монопольный член исчезает, а квадрупольный $\sim Q R_2^2 / r^3 \sim 10^{-8}$ В много меньше $V/2 = 0.5$ В. (Тест в промежутке между сферами намеренно не выполняется, так как МоМ с точечными зарядами имеет точность $O(\text{panel\_size}/r)$, а при $r - R_1 = 0.08$ м $\approx 2$ размера панели погрешность достигает 640%)

### `TestParallelPlates` — параллельные пластины

**Аналитика.**
Для бесконечных пластин (или при $L \gg d$) поле равномерное, $E = V/d$, ёмкость:

$$C_\infty = \frac{\varepsilon_0 L^2}{d}$$

Для конечных пластин с $L/d = 20$ поправка Кирхгофа первого порядка:

$$C \approx C_\infty \left[1 + \frac{d}{\pi L}\left(\ln\frac{\pi L}{d} + 1 + \ldots\right)\right]$$

При $L/d = 20$ относительная поправка составляет $\sim 2$–$5\%$. Из-за концентрации заряда
на краях пластин МоМ даёт $C > C_\infty$.

**Тесты:**
- `test_capacitance_large_aspect_ratio` — при $L/d = 20$ отношение $C_\text{МоМ}/C_\infty$ лежит в диапазоне $(1.0,\; 1.25)$
- `test_charge_neutrality` — полный заряд системы равен нулю
- `test_charge_sign` — пластина с высоким потенциалом несёт положительный заряд
- `test_capacitance_scales_with_area` — при увеличении площади в 2 раза ($\sqrt{2}$ → $L$) ёмкость удваивается с допуском $\pm 15\%$

### `TestPotentialMatrix` — свойства матрицы коэффициентов

- `test_symmetry` — матрица $P$ симметрична ($P_{ij} = P_{ji}$): следует из взаимности Грина
- `test_positive_definite` — все собственные значения $P$ положительны: это гарантирует единственность решения и соответствует положительности электростатической энергии
- `test_diagonal_dominance_sign` — диагональные элементы (самопотенциал панели) максимальны в своей строке: самопотенциал конечного заряда превышает его влияние на удалённые панели при достаточно малых размерах панелей
